La documentation ce trouve dans le répertoire doc :
	-html pour doxygen
	-latex pour doxygen
	-presentation pour les élements de présentatations (PDF, UML, Gantt)

Le code ce trouve dans le repertoire src :
	-versionTxT pour la version texte du jeu, un Makefile ce trouve dans le repertoir. La commande Make suffit pour compiler
	-monopoly_versionSDL pour la version SDL

La seule librairie utilisée dans ce projet est celle SDL2.

	-Le tirage du dé
	-Déplacement du pion
	-Achat ou non de la case
	-Payer loyer
	-Payer impôt
	-Payer loyer des gares
	-Payer loyer des compagnies
	-Piocher une carte
	-Aller en prison
	-Autre action de case
